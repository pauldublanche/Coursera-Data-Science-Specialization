In this course you will learn how to program in R and how to use R for effective data analysis.
You will learn how to install and configure software necessary for a statistical programming environment and describe 
generic programming language concepts as they are implemented in a high-level statistical language. The course covers 
practical issues in statistical computing which includes programming in R, reading data into R, accessing R packages,
writing R functions, debugging, profiling R code, and organizing and commenting R code. Topics in statistical data
analysis will provide working examples.
